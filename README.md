# Starwars-Search

![Starwars-Search Screenshot](https://i.imgur.com/zwbOPI9.png)

Angular 7 + Bootstrap project with Star Wars movie search, character information and history. Retrieves information directly from the [Star Wars API](https://swapi.co/).

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.7 and its versioning managed with Git Flow. Some code taken from the Angular tutorial [Tour of Heroes](https://angular.io/tutorial).

## How to test

1. Navigate to `\starwars-search` and run `npm install` to install all missing dependencies.

2. Run `ng serve` to run the test server. Navigate to `http://localhost:4200/`.

## Requirements and Dependencies

1. [NPM](https://www.npmjs.com/) and [Angular CLI](https://cli.angular.io/)
2. [Angular 7](https://angular.io/)
3. [Bootstrap 4.3.1](https://github.com/twbs/bootstrap/releases/download/v4.3.1/bootstrap-4.3.1-dist.zip)
